## 1. Quick start (printed insert)
1. Connect the two halves with the short USB-C cable.
2. Connect the left half to your computer with a USB-C cable.
3. Wait a few seconds — the per-key displays light up and show characters.
4. Start typing. Press the language key **[ 🌐 ]** to switch input method/layout.
5. For the full manual, scan the QR code / visit **[TO CONFIRM: URL]**.

Safety, first: read the **Safety Notice** before use. Indoor use only, powered from USB 5 V.

## 2. What it is
PolyKybd Split72 is a wired, split mechanical keyboard for PCs, with a small OLED display on every key,
addressable RGB lighting and hot-swappable MX-compatible switches. It connects and is powered entirely
over USB-C (5 V) — no battery, no wireless.

## 3. In the box
- Left and right keyboard halves (assembled version), **or** the kit parts (kit version — see §8).
- Short USB-C to USB-C cable (half-to-half). Host cable [TO CONFIRM if included].
- Printed quick-start with QR to this manual.

## 4. Using the keyboard
- **Connect:** halves together, then left half to the host (see Quick start).
- **Displays:** each key shows its current legend; legends change with layer/language.
- **Language / Unicode:** press **[ 🌐 ]** then choose your OS input method (e.g. `[Win]`, `[Mac]`,
  `[Lnx]`, `[WinC]`). See the firmware documentation for details.
- **Layers & remapping:** the keyboard runs QMK firmware; remap and configure per the QMK docs.

## 5. Care and cleaning
- Disconnect before cleaning. Wipe with a dry or slightly damp (not wet) cloth. No solvents.
- Keep liquids away; the keyboard is not water-resistant. If it gets wet, disconnect and let it dry fully.

## 6. Safety summary
Read the full **Safety Notice**. Key points: indoor dry use only; USB 5 V only; contains small parts
(not a toy — keep from children under 3); handle metal edges with care; keep liquids away; modifications
(kit) void CE conformity. See [CE-Safety-Notice.md](CE-Safety-Notice.md).

## 7. Disposal (WEEE)
Do not discard with household waste. Take it to an electrical-waste collection point (crossed-out
wheelie-bin symbol). Packaging should be recycled per local rules.

## 8. Kit version — building a compliant product
If you bought the kit, the product only meets its CE conformity when built as specified. Assembly is
**mechanical only** — inserting hot-swap switches and keycaps and connecting the FPC cables; **no
soldering is required** for the shipped product. Follow the official build guide (assembly order, ESD
care, correct parts and firmware):
- Build guide: `README.md` in the project repository.
Do not substitute non-equivalent parts or modify the design — doing so voids conformity and may affect
safety/EMC.
